# HSK App - Development Roadmap

## Mục tiêu

Phát triển ứng dụng học HSK đơn giản, dễ sử dụng và hiệu quả cho việc học 5–10 phút mỗi ngày.

Mỗi Phase phải:

- Hoàn thành độc lập.
- Không phá vỡ tính năng cũ.
- Có thể sử dụng ngay sau khi hoàn thành.

---

# Phase 1 - Review & Refactor

## Mục tiêu

Ổn định nền tảng của dự án.

## Công việc

- Review toàn bộ source code.
- Refactor các file lớn nếu cần.
- Chuẩn hóa cấu trúc project.
- Loại bỏ code trùng lặp.
- Cải thiện khả năng mở rộng.
